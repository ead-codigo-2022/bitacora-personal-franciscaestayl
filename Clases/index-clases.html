Clase 2 - HTML

En esta sesión se habló sobre el concepto HTML, o Lenguaje de Marcador para Hipertexto. También se explicó lo que es un "elemento", su estructura y componentes, los cuales son las partes que componen un documento HTML.

Se nos introdujo a la plataforma CodePen para ir trabajando en clases en paralelo a la explicación de los contenidos. Desde ahí pudimos introducir distintas etiquetas y atributos para formar un documento HTML (modo visualización), con imágenes, textos y títulos.

Objetivos de la clase:

  • Introducción a HTML
  • Elementos, etiquetas y atributos
  • Estructura base para un documento HTML
  • Estructura para imágenes y multimedia, y URL

Clase 3 - CCS

En clases vimos el concepto CSS, o Cascading Style Sheets u Hojas de estilos en cascada. CSS es el lenguaje que maneja el diseño y la presentación de las páginas web, el diseño de cómo será la página web al visitarla. Este trabaja en conjunto con HTML. El HTML se encarga del contenido básico de la página, mientras el CSS se encarga de darle estilo a la página web.

Objetivos de la clase:

  • Introducción a CSS
  • ¿Cómo escribir CSS?
  • Recomendaciones de escritura
  • ¿Cómo asociar una hoja de estilos a un documento HTML?
  • Selectores CSS
  • Especificidad y Herencia

Clase 4 - Tipografía y Modelo de caja

En esta sesión se introdujo el uso de tipografías, el cómo descargarlas desde la web. Para la construcción de un documento HTML, se vio cómo usar distintos tamaños y espaciados para los textos, la alineación para la construcción de párrafos y el uso de colores en distintos elementos de un documento. Por último, se nos introdujo al concepto de Modelo de caja, que consiste en entender que todo lo que está contenido en una página tiene la expresión de una caja, comprender estas cajas es clave para poder crear diseños con CSS.

Al final de la sesión se desarrolla un ejercicio práctico, aplicando los conceptos abordados en esta clase y en clases anteriores.

Objetivos de la clase:

  • Uso de tipografía para sitio web
  • Tamaño y espaciado de texto
  • Alineación
  • Uso de color
  • Modelo de caja

Clase 5 - Diagramación

En esta clase vimos cómo se diagrama un documento HTML. Desde el comienzo de la clase trabajamos en un ejercicio práctico el cual consiste en intentar diagramar una página (copiar una presentada) a partir de los conceptos que hemos aprendido de las clases anteriores. Para llegar al resultado final, se fue preguntando por sugerencias de elementos para la construcción de la diagramación, al igual que investigar por cuenta propia en el navegador.

Luego se mostró, a partir de ese mismo ejercicio práctico, maneras de diagramación a partir de grillas.
